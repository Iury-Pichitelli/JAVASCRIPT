> DOM: É um acrônomo para Document Object Model, que esta dentro do navegador.

> Árvore DOM: Ela começa da raiz dentro do navegador sendo o Objeto WINDOW.
Dentro do objeto WINDOW temos outros objetos sendo alguns deles:

* window:
    * Location: Que diz qual é a URL, qual é a página atual, qual é a página anterior;
    * Document: Que é o documento atual;
       * HTML: (parent of Head and Body) and child of Document
            * Head (child of HTML)
                * meta (child of head)
                * title (child of head)
            * Body (child of HTML)
                * p (child of Body)
                    * strong (child of p)
                * div (child of Body)
                * h1 (child of Body)
    * History: Que vai guardando de onde você veio, para onde você vai;

===========================================================================================================

> 5 metodos para acessar os elementos em um DOM:

    * Por marca = tagName
        - getElementsByTagName();
    * Por ID = ID
        - getElementsById()
    * Por Nome = ?
    * Por classe = Class
    * Por seletor = CSS

OPERADORES:

> Aritméticos
        
    5 [+] 2 = 7 { soma }
    5 [-] 2 = 3 { subtração }
    5 [*] 2 = 10 { multiplicação }
    5 [/] 2 = 2.5 {Divisão } -- DIVISÃO REAL, QUE RETORNA VALORES DECIMAIS --
    5 [%] 2 = 1 { Resto da divisão } -- 5 | 2 = % 1 = DIVIDE ATÉ NÃO PRECISAR DE USAR A VÍRGULA --
    5 [**] 2 = 25 { Potência } = 5^2 = 25

     Precedência aritmética:

    1. Parênteses = ()
    2. Potenciação = **
    3. Multiplicação = *
       Divisão = /
       Resto da divisão = %
    4. Adição e subtração = + - 

> Atribuição 

    Auto atribuição

    var n = 3
    n = n + 4
    ( n vale 7 agora, pois é necessário tirar o valor na memória da var inicial e atribuir o novo valor)

    Encurtando uma atribuição

    var n = 3
    n = n + 4 (encurtando fica n += 4)
    n = n - 5 (encurtando fica n -= 5)
    n = n * 4 (encurtando fica n *= 4)
    n = n / 2 (encurtando fica n /= 2)
    n = n** 2 (encurtando fica n**= 2)
    n = n % 5 (encurtando fica n %= 5)

> Incremento

    var x = 5
    x = x + 1 (Simplificando x++)
    x = x - 1 (Simplificando x--)

    \ Pre-incremento 
      --x ou ++x

    \ Pos-incremento
      x-- ou x++

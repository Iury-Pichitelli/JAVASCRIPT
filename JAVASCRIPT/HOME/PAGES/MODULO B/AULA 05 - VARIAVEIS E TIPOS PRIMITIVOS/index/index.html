> Variáveis:

  palavra "let"

===========================================================================================================

> A três formas de declarar uma string em javascript com "", '' ou ``;

===========================================================================================================

> Indentificador = nome da variável.

  Regras de identificadores:

  Podemos começar com letra, $ ou _
  É possível usar letras ou números
  É possível usar acentos e símbolos
  Não podem começar com números
  Não podem conter espaços
  Não podem ser palavras reservadas do JS

===========================================================================================================

> No javascript os números não são separados do tipo inteiro e decimnal (int ou float)
  Todos são (number) = 5, 18, -12, 0.5, -15.9, 3.14, 8.0
  Valor boleano = boolean = False or True

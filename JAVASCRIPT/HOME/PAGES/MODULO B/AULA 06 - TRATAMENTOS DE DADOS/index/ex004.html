Comandos:

Formatação de strings

var nome = window.prompt('Qual é o seu nome?' )
document.write(`Seu nome tem ${nome.length} letras`)

document.white() = escreve no html do site
document.whiteln() = escreve no html mas em uma nova linha

Podemos colocar tags dentro das crases.

Ex: 
document.writeln(`Olá strong> ${nome} /strong> Seu nome tem ${nome.length} letras`)
document.write(`Olá ${nome}, seu nome em letras maiúsculas é ${nome.toUpperCase()}`)

================================================

Formatação de números:

var n1 = 1545.5
document.write(`Seu numero com duas casas decimais: {n1.toFixed(2)}`)

Para trocar o ponto por vírgula podemos fazer o seguinte Comandos
document.write(`Seu numero com duas casas decimais: {n1.toFixed(2).replace('.', ',')}`)

================================================

Exemplo:

var num = window.prompt('Digite um número decimal: ');
var quant = window.prompt('Quantos casas decimais voce quer? ');

window.alert(`ola ${Number(num).toFixed(Number(quant)).replace('.', ',')}`);

================================================

Para converter a moeda:

n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})

Regra de precedência dos operadores:
                
Primeiro aritméticos: [ () **  *  /  %  +  - ] 
Segundo relacionais:  [ >  <  >=  <=  ==  !=  === ]
Terceiro lógicos:     [ !  &&  || ] = not, and e or
Quarto ternário:      [ ?  :]
            

OPERADORES:

> Relacionais

    Ordem de precedência: Da esquerda para a direita

    [ > ] = maior que 
    [ < ] = menor que
    [ >= ] = maior ou igual que
    [ <= ] = menor ou igual que
    [ == ] = igual
    [ != ] = diferente
            
Para toda expressão que tenha um operador relacional, o resultado sempre será um valor boolean
(true or false)
    
    5 [ >  ] 2 = true
    7 [ <  ] 4 = false
    8 [ >= ] 8 = true
    9 [ <= ] 7 = false
    5 [ == ] 5 = true
    4 [ != ] 4 = false

    Primeiro são os operadores aritméticos e depois os Relacionais

    ex: a <= b - 10

    ================

    Identidade '==='

    5 [ == ] 5 = true 
    5 [ == ] '5' = true
    5 [ === ] '5' = false ( O JS não verifica o tipo com '==' mas sim com '===' )

    Diferente estrito

    [ !== ] = são de tipos diferentes

===========================================================================================================

> Lógicos
    
    Ordem de precedência = NOT, AND e OR

    [ !  ] = negação = not
    [ && ] = conjunção = and
    [ || ] = disjunção = or

    ================================

    Operador unário (um operando)
    [ !  ] = negação = not
    false = true
    true = false

    ================================

    Operador binário (dois operando)
    [ && ] = conjunção = and

    true && true = true
    true && false = false
    false && true = false
    false && false = false

    ================================

    Operador binário (dois operando)
    [ || ] = conjunção = or

    true && true = true
    true && false = true
    false && true = true
    false && false = false

===========================================================================================================

> Ternário (precisa de três operandos)

    [ ? ] se for
    [ : ] se não

    teste [ ? ] true [ : ] false

    média >= 7.0 [ ? ] 'Aprovado' [ : ] 'Reprovado'

    var x = 8
    var resultado = x % 2 == 0 ? 5 : 9
    x = 8
    resultado = 5
